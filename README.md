## Environment

- Deployed on [Vercel](https://vercel.com/).
- Using `Node.js v18.18.2` with `asdf`.
- React v18.2
- Next.js v13.5
- TypeScript v5.2
- Chakra UI v2.8
- Swiper v10
- react-icons


## Startup local server

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
